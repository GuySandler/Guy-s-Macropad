# Blueprint1_hackpad

The smallest macropad I could make with a screen and 8 neopixels,

# BOM:
- 1x XIAO RP2040
- 8x Cherry MX switches
- 1x 128x32 OLED Display
- 8x SK6812 MINI-E LEDs
- 8x DSA keycaps
- 4x M3x16 Bolt
- 4x M3 Heatset
